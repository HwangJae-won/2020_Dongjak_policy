# 2020 동작구 빅데이터 활용 정책 아이디어 공모전
# 1. 개요
- 주제: 스코어링모델과 데이터시각화를 이용한 금연거리 및 흡연구역 위치 선정 
- 기간: 2020.03 ~ 2020.05
- 팀원: 임유경, 정서연, 황재원
- 사용 skill: R, SPSS

동작구에서 주최한 [2020 빅데이터 활용 정책 아이디어 공모전]에 참가하여 데이터를 활용한 문제 제시와 정책을 고민하던 중 길거리의 간접흡연 문제가 심각하다는 점, 보건복지부의 금연종합대책 등을 참고하여 적절한 **금연거리 및 흡연구역의 위치를 제안**하는 프로젝트를 진행하였습니다. 동작구 내 우선 배치 행정동을 선정한 후 금연거리 및 흡연 구역 위치를 제안하였으며 분석툴은 R과 SPSS을 사용하였습니다.

# 3. 금연거리 및 흡연부스 우선배치 행정동 선정
## 3-1) 데이터 전처리
금연 거리 및 흡연 구역을 선정하는 데 있어** 1) 간접흡연 피해 추정 데이터  2) 흡연자 수 추정 데이터** 의 두 가지 요인을 선정하여 이와 관련된 데이터를 수집하였습니다. 수집한 데이터셋은 단위가 상이하였기에 합계를 구한뒤 행정동 단위로 나누어 비율로 나타내었습니다.<img width="1179" alt="image" src="https://user-images.githubusercontent.com/79994991/166674069-a32b2a7a-2e4c-47ed-b186-841e2337c6f7.png">


## 3-2) 분석
지도에 만들어 낸 feature를 시각화하여 필요성을 확인하였고 가중치와 변수값을 이용해 행정동별 스코어를 구한 후 0~5점으로 scaling해주는 scoring model을 활용하여 각 feature에 가중치를 부여하였습니다. 가중치를 반영하여 상위 4개의 행정동을 우선 배치 행정동으로 선정하였습니다.

# 4. 금연거리 및 흡연구역 선정
## 4-1) 데이터 전처리
우선배치 행정동을 선정할 때와는 다르게 행정동 별로 분류하여 비율을 고려할 필요가 없었으므로 간단한 데이터 전처리 과정만 거쳤습니다. 지하철과 버스 승하차 인구를 대중교통 데이터로 통합하였고, 다양한 시설 중 흡연에 좀 더 영향을 받을 것으로 예상되는 취약 계층 시설을 선별하였습니다.

## 4-2) 분석
고려할 변수들을 R의 ggplot2를 활용하여 시각화한 결과 변수들이 집중되어 있는 곳을 육안으로 판단하여 직접 탐사를 통해 적합한 금연거리를 제안하였습니다.

## 5. 한계점
기초 지식이 많이 부족한 상태로 처음 도전해본 공모전이었기에 많이 부족한 점이 있었지만 데이터를 바라보고 시각화해보며 데이터의 실용성을 체감해볼 수 있었습니다. 또한 QGIS 등의 지리적 시각화 툴을 활용하였다면 좀 더 정확하고 다양한 시각화를 진행할 수 있었을 것 같다는 아쉬움이 남습니다.
